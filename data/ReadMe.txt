<GWAS>
These files are generated using the files that are downloaded from the GWAS Catalog. (https://www.ebi.ac.uk/gwas/docs/file-downloads)
Please use these files only for testing the classifier. When you use these data, please cite the GWAS Catalog.
==> MacArthur, Jacqueline, et al. "The new NHGRI-EBI Catalog of published genome-wide association studies (GWAS Catalog)." Nucleic acids research 45.D1 (2016): D896-D901.
